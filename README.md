
# 🎬 Netflix Clone - Full Stack Project

Build with **React** (Frontend), **Spring Boot** (Backend), and **MySQL** (Database).

## 📂 Folders
- `netflix-backend`: Spring Boot API Source
- `netflix-clone`: React UI Source

## 🚀 Quick Setup
1. **Database:** Create `netflix_db` in MySQL.
2. **Backend:** Run `mvn spring-boot:run`
3. **Frontend:** Run `npm run dev`
